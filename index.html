Install
Up and running in 60 seconds.

No coding required. Just 4 clicks.

1

Clone or download the repo

Go to the GitHub repo and click Code → Download ZIP, or clone with git clone

2

Unzip the file

Right-click the ZIP → Extract All (Windows) or double-click (Mac).

3

Open Chrome extensions

Go to chrome://extensions/ and turn on Developer mode (top-right toggle).

4

Load the extension

Click "Load unpacked" → select the unzipped folder. The green Zenith shield appears in your toolbar — you're protected!
