# Zenith AdBlocker Privacy Policy

Zenith AdBlocker processes blocking rules locally on the user's device.

Zenith does not sell browsing data.

Zenith does not collect page content.

Zenith does not send visited websites to a remote server.

Zenith stores extension settings locally, including:
- enabled/disabled status
- whitelist domains
- blocking counters
- local blocked request logs
- filter list settings

Blocked request logs are stored locally only and can be cleared by the user.

Zenith may download text-based filter lists from trusted sources to improve blocking quality. These lists are treated as data, not executable code.
